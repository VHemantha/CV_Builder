# Social Media Images for SEO

## Required Images

### 1. Open Graph Image (og-image.png)
- **Size**: 1200 x 630 pixels
- **Format**: PNG or JPG
- **Purpose**: Shown when sharing on Facebook, LinkedIn
- **Content**: CV Builder logo + tagline "Free Professional Resume Builder"

### 2. Twitter Card Image (twitter-image.png)
- **Size**: 1200 x 600 pixels
- **Format**: PNG or JPG
- **Purpose**: Shown when sharing on Twitter/X
- **Content**: CV Builder logo + tagline

### 3. Favicon (favicon.ico)
- **Size**: 32 x 32 pixels (and 16x16)
- **Format**: ICO
- **Purpose**: Browser tab icon
- **Content**: CV Builder logo icon

### 4. Apple Touch Icon (apple-touch-icon.png)
- **Size**: 180 x 180 pixels
- **Format**: PNG
- **Purpose**: iOS home screen icon
- **Content**: CV Builder logo

## Design Guidelines

1. **Brand Colors**:
   - Primary: #4285f4 (Blue)
   - Secondary: #667eea (Purple)
   - Background: White or gradient

2. **Typography**:
   - Clear, readable fonts
   - High contrast text
   - Professional appearance

3. **Content**:
   - CV Builder logo/name
   - Tagline: "Free Professional Resume Builder"
   - Optional: Document icon or resume visual

4. **Requirements**:
   - High quality (not pixelated)
   - Under 1MB file size
   - Optimized for web

## Tools for Creating Images

- Canva (free templates available)
- Figma (design tool)
- Adobe Photoshop
- GIMP (free alternative)
- Online generators: realfavicongenerator.net
